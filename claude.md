# 開発の進め方
開発はサブエージェントを5つ使って役割分担をして開発を進めてください。
## エージェント構成
1.PM Agent
2.Designer Agent
3.Frontend Agent
4.Backend Agent
5.QA Agent

## 進め方詳細
- まずPM Agentがタスク分解→各エージェントに作業を並列ディスパッチ。依存関係がある場合は先行タスクが完了してから、先行タスクの完了状況を含めてディスパッチ
- 各エージェントは独立して作業実行。依存関係があり先行タスクが完了していない場合はPM Agentに打ち返す
- 各エージェントは自分がやるべきではない作業（例えばFrontendの修正中にBackendの修正が必要なことに気づいた、など）はPMに打ち返す。PMはタスクの追加を検討し、追加が必要なら追加してディスパッチする
- 完了次第、結果をマージして最終成果物として出力

進捗バーで各エージェントのステータス（実行中/完了）を表示しながら進めて
